# SubliMaster

SubliMaster یک سیستم ERP و حسابداری جامع جهت مدیریت سفارشات، موجودی پارچه، امور مالی، چک‌ها و مشتریان برای کارگاه‌های چاپ سابلیمیشن است. 

## امکانات واقعی پیاده‌سازی شده
- ثبت سفارشات چاپ سابلیمیشن با محاسبه خودکار متراژ و قیمت
- کسر و برگشت خودکار موجودی پارچه هنگام ثبت و ویرایش سفارش
- اتصال کامل مانده حساب و بدهی مشتریان به فاکتورها و پرداخت‌ها
- سیستم مدیریت کامل چک‌ها
- پشتیبان‌گیری ابری و لوکال خروجی Room Database (.db)

## امکانات ناقص یا محدودیت‌ها
- خروجی PDF فعلی به دلیل پشتیبانی نکردن کلاس پیش‌فرض از فونت، فقط از کاراکترهای انگلیسی/لاتین پشتیبانی می‌کند (متون فارسی در PDF ظاهر نمی‌شوند).
- خروجی گزارشات فعلاً برای لیست سفارشات فرمت CSV تولید می‌کند نه xlsx واقعی. 
- امکان گزارش‌گیری بازه‌های زمانی (از تاریخ تا تاریخ) پیاده‌سازی نشده است.

## نحوه Compile و Build گرفتن
جهت اجرای پروژه، از دستورات استاندارد Gradle به شرح زیر استفاده کنید:

```bash
# بیلد نسخه دیباگ
./gradlew clean assembleDebug
```

مسیر فایل APK پس از Build: 
`app/build/outputs/apk/debug/app-debug.apk`

## نحوه بکاپ گیری و بازیابی
۱. در اپلیکیشن وارد بخش **تنظیمات / بکاپ** شوید. 
۲. روی "**تهیه فایل پشتیبان**" کلیک کنید تا فایل `sublimation_db` توسط Storage Access Framework گوگل (در موبایل یا درایو) ذخیره شود.
۳. در دستگاه جدید، روی "**بازیابی از فایل**" کلیک کرده و همان فایل را انتخاب کنید. مسیر ارتباط با دیتابیس فوراً بسته شده و فایل جدید جایگزین می‌شود. سپس برنامه را مجددا اجرا کنید.
